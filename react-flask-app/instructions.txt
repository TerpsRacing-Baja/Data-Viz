Requirements:
- Venv
- PostgresSQL application
- psycopg2 (only downloadable if you have postgres)
- react-scripts

Terminal 1:
cd to file/api

//If on Linux
 python3 -m venv venv
source venv/bin/activate
yarn start-api

//If on windows
 (cd venv\bin && set FLASK_APP = api.py && set FLASK_ENV=development && flask run if windows) (this is optional/may not change/do anything.)
 $ python -m venv venv
 $ venv\Scripts\activate
 $ pip install flask python-dotenv
 $flask run (may need to rename api.py to app.py)


Terminal 2:
cd into react-flask-app
yarn start (may need to npm install react-scripts --save)
