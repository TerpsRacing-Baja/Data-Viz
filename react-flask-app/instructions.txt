Requirements:
- Venv
- PostgresSQL application
- psycopg2 (only downloadable if you have postgres)
- react-scripts
- yarn
- flask (pip install flask python-dotenv)
- pandas

#STARTING VENV IS OPTIONAL BUT PREFERED
python3 -m venv venv
source venv/bin/activate (venv\Scripts\activate if on windows)

Terminal 1:

//If on Linux
yarn start-api

//If on windows

 cd into react-flask-app/flask
 $flask run (may need to rename api.py to app.py)


Terminal 2:
cd into react-flask-app/react
yarn start (may need to npm install react-scripts --save)
